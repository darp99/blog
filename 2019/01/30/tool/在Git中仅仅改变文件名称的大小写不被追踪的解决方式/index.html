在Git中仅仅改变文件名称的大小写不被追踪的解决方式

Jan 30 2019 tool

在对文件命名大小写不敏感的文件系统中，如果你改了文件名（只是大小写变化），Git 默认模式是识别不了这种变化的，自然就无法提交，那该怎么做呢？有不少方法，看看 StackOverflow 的讨论。当然绕过这个问题的办法是约定所有的文件名小写。

git mv -f OldFile oldfile
